# HomeCare3
HomeCare

Este é um projeto pessoal que tem como intuito desenvolver um sistema simplificado e de baixo custo para administração de agendamentos de prestação de serviços terceirizados,
conectando o usuário com o WhatsApp do atendimento da empresa X,
enviando uma mensagem pré-formatada com todos os dados necessários para realização do atendimento (p.e.: endereço, cpf, telefone.)

/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*/*

O objetivo deste aplicativo é ser simples e barato,
porém a versão final pretendida ainda contará com: 

-Filtros de CEP e CPF;
-API de pagamento online (eCommerce);
-Banco de Dados para controle de Usuários tipo Cliente e tipo Prestador de Serviços;
-Lista Dinâmica do tipo RecyclerView para apresentação;
